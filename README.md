# Mountain Trek - Professional Trekking Guide Website

A modern, responsive website for Sandesh Baral's professional trekking guide services. This website showcases trekking destinations, services, and allows potential customers to get in touch.

## Features

- Responsive design that works on all devices
- Modern animations and transitions
- Interactive contact form
- Testimonials section
- Popular destinations showcase
- Mobile-friendly navigation

## Setup Instructions

1. Clone this repository to your local machine
2. Add the following images to the `images` folder:
   - hero-bg.jpg (A stunning mountain landscape for the hero section)
   - guide-profile.jpg (Your professional profile photo)
   - everest.jpg (Everest Base Camp trek photo)
   - annapurna.jpg (Annapurna Circuit trek photo)
   - langtang.jpg (Langtang Valley trek photo)
   - testimonial1.jpg, testimonial2.jpg, testimonial3.jpg (Testimonial profile photos)

## Technologies Used

- HTML5
- CSS3 with modern features (Grid, Flexbox, Variables)
- JavaScript (ES6+)
- AOS (Animate On Scroll) library
- Font Awesome icons

## Customization

You can easily customize the website by:

1. Modifying the color scheme in the CSS variables (in styles.css)
2. Adding/removing destinations in the destinations array (in script.js)
3. Updating testimonials in the testimonials array (in script.js)
4. Changing the content in index.html

## Contact Form Integration

The contact form is currently set up to show a success message. To make it functional:

1. Create a server endpoint to handle form submissions
2. Update the form submission code in script.js to send data to your server
3. Add proper form validation and error handling

## Browser Support

The website is compatible with:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
